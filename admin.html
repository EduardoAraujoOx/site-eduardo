Como usar

  1. Adicione, edite ou exclua publicações usando os botões abaixo
  2. Use o checkbox "Fixar" para destacar publicações importantes (aparecem primeiro)
  3. Clique em "Exportar JSON" para baixar o arquivo atualizado
  4. Substitua o arquivo data/publications.json pelo arquivo baixado
  5. Faça commit e push das alterações para atualizar o site
