Metodologia
  • Fonte: SICONFI (STN) — RREO Anexo 3, coluna "Total Últimos 12 Meses". Para 2019–2025: período 6 (bimestre nov–dez). Para 2026: período 2 (bimestre mar–abr, dado mais recente disponível). Acesso via API Data Lake do Tesouro Nacional.
  • ISS: soma dos municípios (co_esfera=M), conta ISSLiquidoExcetoTransferenciasEFUNDEB. ISS ES = somatório dos 78 municípios capixabas.
  • ICMS: conta ICMSLiquidoExcetoTransferenciasEFUNDEB de cada estado (co_esfera=E), líquido de transferências e FUNDEB.
  • Deflator: IPCA acumulado anual (IBGE), com base em dezembro de 2025 (preços jan/2026).
  • Coeficiente: Total A (ES, soma da série deflacionada) / Total B (BR, soma da série deflacionada), conforme metodologia da LC 214/2025.

¹ Dados de ISS para o Brasil exigem consulta individual a 5.570 municípios — coletados via GitHub Actions (27 jobs paralelos por estado). ISS BR poderá exibir "aguardando" até conclusão da primeira coleta.

² Valores nominais em R$ bilhões. Atualização monetária pelo IPCA/IBGE.

Fonte: SICONFI — API Data Lake STN. Script de coleta disponível em /data/collect-siconfi.py.
